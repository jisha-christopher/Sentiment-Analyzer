# 😊 Sentiment-Analyzer

A simple web-based Sentiment Analysis project built using HTML, CSS, and JavaScript.  
It analyzes input text and predicts whether the sentiment is Positive, Negative, or Neutral.

## 📌 Features
- Simple and clean UI
- Real-time text analysis
- Detects sentiment of input text
- Lightweight and fast
